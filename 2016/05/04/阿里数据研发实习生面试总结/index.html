首先，只是一篇走心的倒叙的心路历程。

拿到录用函到通知体检的3个星期的神游阶段：

小情绪：人家都已经给能去阿里，还不如让人家去实习。。。。嘤嘤嘤。。。每天焦虑等待着，把当下的事情都忽略了。惭愧。
才华：理智的一面，其实就剩论文了，5月杭州，看来的等等到秋天才回来。
在阿里出了结果拖了很久，中间错过了很多其他的机会，大部分都拿有保底，也是有点冒险。毕竟阿里今年实习生招聘的太早了。
走心的套路应该是：做好当下事情。
不过，最终，才华战胜了情绪。

后面实习延续写实习日志的优秀习惯，先mark。

面试阶段：

倒序回忆，共5面.(每次面完很激动)

小管家体检通知（5.3上午）
正式offer (4.15下午)
待跟进offer(4.14下午)

等了26天啊 。。还好没有放弃。

第五面 hr面（3.18下午）

超级温柔的hr

  1. 你怎么看待去阿里实习？
  2. 讲一下最近的项目?
  3. 你有什么要问我的吗？

第四面 交叉面（3.18上午8：30左右）交叉面：不是你应聘部门的面试官的加面。

也是特别温和

  1. 自我介绍
  2. 有没有有上线的项目或者UV访问量特别大的产品？
  3. 技术你还会哪些？
  4. 有什么要问我的嘛
    （基本就是基于你答的地方开始深问）

第三面 部门老大（3.14上午）

语气正常，问的问题比较宏观。

  1. 自我介绍
  2. 你理解的数据是什么
  3. 谈谈你的牛小招打算怎么做数据分析？
  4. E－R图是什么？
  5. 你有什么用问我的吗？

第二面，部门主管（3.11下午）

压力面，抗了一个小时，面傻了要，人生中第一次压力面。
考察四个方面（大数据相关的研发，数据分析能力，产品化能力，抗压）

  1. 自我介绍
  2. 你没有数据研发相关的经验，你做个学习规划？
  3. 去年天猫双十一，交易额945亿，今年要达到1200亿，要怎么达到？（涉及到，用户细分，用户调研，建模能力，产品需求。。）
  4. 给了一个场景，自己提出不足，自己想法改进。
  5. 假如deadline要到了，你咋办呢？
  6. 有什么要问我的嘛？

第一面 数据PD（3.11上午）

正常语气

  1. 自我介绍
  2. 讲一下你的牛小招这款产品？
  3. 你在这个里面做了哪些事情？

  4. 你有什么问题要问我的嘛？

    ###面试之前，我准备了什么？

  5. 产品回答总结，约1w字左右。（本来是应聘产品岗，后面岗位不招了最后转到数据研发岗位）
  6. 和小伙伴组织模拟面试。（针对你的简历还有你的自我介绍提问）
  7. 简历修改了4，5遍的样子。

建议：

  1. 说话要清楚，很有可能上午面完，下午就打过来了，所以保持电话通畅。
  2. 自我介绍是重点，我的大部分都是由此发问。
  3. 针对面试官的问题，最好提前准备，面试官也是可以感受到你的水平的。
  4. 中间的过程很漫长，耐心等待。
  5. 条件允许，可以找同学做模拟面试，保持状态。
  6. 如果是面试官私人电话打过来的，加个微信，如果面试官愿意的话，后面有问题会很方便。
